# 🛍️ ShopEase - Full Stack E-Commerce Application

ShopEase is a modern Full Stack E-Commerce web application built using **React.js** and **Spring Boot**. 
It provides a seamless shopping experience with secure authentication,
product browsing, cart management, order placement, and an admin dashboard.

---

## 🚀 Features

### 👤 User
- User Registration & Login
- JWT Authentication
- Browse Products
- Search & Filter Products
- View Product Details
- Add to Cart
- Update Cart Quantity
- Remove Products from Cart
- Checkout
- Place Orders
- Order History
- Responsive Design

### 🛠️ Admin
- Admin Login
- Add Products
- Update Products
- Delete Products
- Manage Inventory
- View Customer Orders

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/shopease-fullstack.git
```

---

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

Runs on:

```
http://localhost:8080
```
