# 🛍️ ShopEase - Full Stack E-Commerce Application

ShopEase is a modern full-stack e-commerce application built using **React.js**, **Spring Boot**, and **MySQL**. It provides a secure and responsive online shopping experience with JWT-based authentication, product management, shopping cart, and order processing.

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Product Browsing & Search
- Shopping Cart
- Order Management
- Admin Dashboard
- Responsive UI

## 🛠️ Tech Stack

**Frontend:** React.js, Vite, JavaScript, HTML, CSS

**Backend:** Java, Spring Boot, Spring Security, Spring Data JPA, Hibernate

**Database:** MySQL

## ⚙️ Getting Started

```bash
git clone https://github.com/roshni-selvam/ShopEase-fullstack.git
cd ShopEase-fullstack
```

**Frontend**

```bash
cd frontend
npm install
npm run dev
```

**Backend**

```bash
cd backend
mvn spring-boot:run
```
